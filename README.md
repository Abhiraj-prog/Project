1--->  # fHandwritten Digit Recognition using ANN

This project uses an Artificial Neural Network (ANN) built with TensorFlow/Keras to recognize handwritten digits (0–9) using the MNIST dataset.

## Key Features:
- MNIST handwritten digit dataset
- Image preprocessing and normalization
- ANN model training
- Digit prediction
- Custom handwritten image testing

## Tech Stack:
Python, TensorFlow, Keras, NumPy, Matplotlib

## Goal:
To classify handwritten digits accurately and understand deep learning fundamentals.

2--> # Customer Churn Prediction using ANN

This project uses an Artificial Neural Network (ANN) to predict whether a bank customer will leave or stay based on customer details such as credit score, geography, gender, balance, and activity.

## Key Features:
- Data preprocessing and encoding
- Feature scaling
- ANN model training
- Customer churn prediction
- Overfitting analysis using loss/accuracy graphs

## Tech Stack:
Python, Pandas, NumPy, Scikit-learn, TensorFlow/Keras, Matplotlib

## Goal:
To build a binary classification model that helps predict customer churn and supports better retention strategies.
